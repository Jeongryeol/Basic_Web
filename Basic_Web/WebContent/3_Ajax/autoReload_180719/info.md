# auto_Reload 사고과정..
#### Ajax를 이용한 웹화면의 일부분을 자동으로 갱신하는 코드 작성하기

#### 이 폴더의 화면은 autoReloadpage.jsp로부터 출발합니다.

>##### 포털사이트에서 각 분야별 최고 인기 뉴스의 링크를 자동으로 계속해서 갱신해주는 코드를 작성합니다. 
>###### 화면을 갱신하는 과정을 자동으로 반복시키되 그 시간 간격을 정해야하고,
>##### 반복되는 내용이 저장되서 바꿔출력하는 처리를 수행할 페이지가 필요합니다.

>##### <BODY>태그의 스크립트영역에 반복시작과 반복정지을 정의한 함수를 호출해두면
>##### 페이지를 열었을때 자동으로 갱신을 시작할 수 있습니다.

>##### 반복시작하는 함수 start()는
>##### "대상함수"를 "지정한 시간(ms)"으로 지연시키면서 반복시키면서 각각의 지연시간을 변수에 저장합니다.
>##### 저장된 변수는 반복을 초기화하는 함수 stop()에서 지연시간이 초기화됩니다.

>##### 반복할 함수는 ajax를 통해서 그때그때의 화면을 갱신합니다.
>##### 원리는 다음과 같습니다. 갱신할 뉴스가 들어있는 별도의 웹페이지(getNewsTitle.jsp)에는
>##### 스크립틀릿을 이용해서 자바코드로 뉴스를 순서대로 담아줍니다.
>##### 담아준 뉴스는 숫자를 카운트할 변수를 디클러레이션( <%! %>)에 선언해두었습니다.
>##### switch문에 이 변수를 사용해서 각 숫자의 경우(case)에 맞추어 뉴스가 바꾸어 출력될 수 있도록 세팅합니다.
>##### 숫자가 n일때 브라우저에 출력할 뉴스를 테이블로 넘겨주고, 숫자 n을 1올립니다(1++,후위연산자)
>##### 마지막 숫자일때 초기번호 숫자로 돌려줍니다.
